# Resource credits

All tile graphics in this directory (`0`–`8`, `unknown`, `marker`, `bomb`,
`bomb_triggered`, `marker_wrong`) are **original artwork**, generated
programmatically from basic shapes plus rendered digits. They are released
under the same MIT license as the rest of this project.

- **Digits** are set in **DejaVu Sans Bold**
  (<https://dejavu-fonts.github.io/>), whose license is free and permissive
  (Bitstream Vera / public-domain-equivalent terms). The font itself is **not**
  redistributed in this repository — only rasterized glyphs baked into the PNGs.
- The tiles are 32×32 PNGs, rendered at 8× and downscaled (LANCZOS) for smooth
  antialiasing.

The app icon (`resources/minesweeper.ico` / `.png`, a bomb on a beveled tile) is
likewise **original artwork**, generated the same way, under the same MIT license.
